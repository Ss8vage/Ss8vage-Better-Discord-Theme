Thanks for downloading Ss8vage ClearVision Edited Better Discord Theme!

Please make sure you have Discord and Better Discord installed.

Discord Download: https://discord.com

Better Discord: https://betterdiscord.app

Tutorial for installing the theme into Discord:
Copy the .css file
Navigate to User Settings > Themes > Open Themes Folder
Paste the .css file into the folder
Go back to User Settings > Themes
Click the gray "X" button to enable it!
